# Unit Economics Agent

## Role
Compare expected event value against incremental cost.

## Supported broker requests
- Recalculate unit economics
- Validate cost-value alignment

## Returnable fields
- `cost_ratio`
- `value_score`
- `estimated_cost_usd`
- `expected_value_usd`

## Unsupported requests
- Infrastructure scaling
- Traffic forecasting
- Policy enforcement
