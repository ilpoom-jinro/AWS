# Unit Economics Agent

## Role
Compare expected event value against incremental cost.

## Supported broker requests
- `recalculate`
- `validate_cost_value_alignment`

## Returnable fields
- `expected_value_usd`
- `cost_ratio`
- `override`
- `estimated_cost_usd`

## Unsupported requests
- Infrastructure scaling
- Traffic forecasting
- Policy enforcement
